# PanApp

PanApp es un aplicativo web diseñado para la gestión integral de una panadería. Permite administrar inventario, productos, ventas, proveedores e insumos de manera eficiente, facilitando el control y organización de los procesos diarios.

Está orientado a negocios de panadería que buscan digitalizar y optimizar su operación.

##  Objetivo
Optimizar y agilizar los procesos administrativos y operativos de una panadería, reduciendo errores manuales y mejorando el control del inventario y las ventas.

##  Funcionalidades
-  inicio de sesión (Login)
-  Registro de usuarios
-  Gestión de roles (Administrador y Cajero)
-  Registro de ventas
-  Gestión de inventario
-  Gestión de productos
-  Gestión de proveedores
-  Gestión de insumos
-  Control de entrada y salida de materia prima

## 🛠️ Tecnologías
- Lenguaje principal: Python
- (Base de datos Laragon)
